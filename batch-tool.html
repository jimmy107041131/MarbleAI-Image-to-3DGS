MarbleAI Image to 3DGS Tool

Convert images to 3D Gaussian Splat (PLY) files using the World Labs Marble API. Designed for designers who need a zero-install browser tool.

Features:

  • Single Image — Drag & drop images, generate 3D worlds, download PLY/SPZ
  • Batch Folders — Watch an input folder, auto-process new images, save to output folder
  • Model selection — Marble 0.1-mini (fast) or Marble 0.1-plus (high quality)
  • Resolution — Full Res / 500K / 100K splat output
  • Flexible outputs — PLY, SPZ, Thumbnail, Marble viewer URL
  • Auto-save — Directly write to a local output folder (e.g. TouchDesigner input)
  • Download All — Pack all results into a ZIP with links.csv
  • Settings memory — All preferences saved locally
Developed by JimmyMg
Powered by World Labs Marble API & spz-js
